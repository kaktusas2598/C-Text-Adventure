# C Text Adventure Game

Fun little experiment building Old School text based adventure game in C, based on tutorial series from https://helderman.github.io/htpataic/htpataic01.html

# Build instructions

 * Run `make object.h`
 * Run `make object.c`
 * Run `make`
 * Run textAdventure.exe
